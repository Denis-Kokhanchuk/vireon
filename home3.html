Курс з UI/UX дизайну для початківців

Цей курс призначений для тих, хто хоче розпочати свою кар'єру в UI/UX дизайні. Протягом курсу учасники отримають базові знання, навички та практичний досвід, які допоможуть створити свій перший проєкт.

Мета Курсу

  • Надати учням базові знання про UI та UX.
  • Навчити використовувати інструменти дизайну (зокрема Canva).
  • Пояснити процес створення інтерфейсів та користувацького досвіду.
  • Допомогти створити власний проєкт.
  • Надати сертифікат після успішного завершення курсу.

Формат Навчання

  • Відео-уроки з поясненням теорії та практичними прикладами.
  • Домашні завдання для закріплення матеріалу.
  • Онлайн-уроки з ментором для розбору складних тем та фідбеку.
  • Фінальний проєкт, який оцінюється ментором.

Ключові Знання

  • Розуміння різниці між UI та UX.
  • Навички роботи з Canva для створення дизайнів.
  • Принципи створення зручних інтерфейсів.
  • Основи юзабіліті.
  • Здатність проєктувати сторінку товару.

Роадмап Курсу

Детальний план навчання, що охоплює ключові етапи UI/UX дизайну.

День Зміст заняття

День 2Основи юзабіліті. Принципи хорошого інтерфейсу. Мобільні vs. десктопні інтерфейси.
День 3-7Практичне заняття — проєктування сторінки товару.
ПроєктСтворення фінального проєкту, оцінка ментором.
